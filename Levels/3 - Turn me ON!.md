#DESAFIO
http://www.deathball.net/notpron/false/movetotheothersite.php

#DICAS
1) Turn me ON / Me acenda
1) url: move to the other site / mude para o outro site
2) gif: stop being so negative! / pare de ser tão negativo!

#SOLUÇÃO
Comumente é usado para determinar como falso a palavra OFF e a dica 1 indica com "Turn me ON" algo deixará o estado OFF para ON ou seja, de false para true.
A dica 2 fala sobre a mudança para outro site que geralmente é feita por url.
Frisando a dica 1, a dica 3 indica para deixar de ser negativo ou seja, ser postivio.

- alterar o url de 'http://www.deathball.net/notpron/false/movetotheothersite.php' para 'http://www.deathball.net/notpron/true/movetotheothersite.php'
